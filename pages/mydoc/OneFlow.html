    • Getting started¶

    Step 1 - Create a Oneflow account

    If you do not already have an account, head over to the Oneflow application and create a trial account.

    Step 2 - Generate an API token

    Once you have your account, go to the extensions page and enable the API token extension.

    When enabled, create an API token and temporarily write down the token you get (you will only see it once).

    Step 3 - Setup authentication

    Read the authentication page and follow the instructions on how to setup authentication.

    Step 4 - Make your first request

    Make a request to the ping endpoint to check your connection and authentication.

    cURL

    curl -X GET 'https://app.oneflow.com/api/ping' \
-H 'Content-Type: application/json' \
-H 'X-Flow-API-Token: API_TOKEN'

    Python

    import requests

api_token = 'xxxx'

headers = {
    'Content Type': 'application/json',
    'X-Flow-API-Token': api_token,
}
r = requests.get('https://app.oneflow.com/api/ping', headers=headers)
print(r.status_code)

    If everything is working properly you should get a HTTP 200 response with an empty JSON object in the response body ({}).

    Step 5 - Build your workflows!

    Now you are ready to start building your automation workflow!

    We suggest that you take a look at the rest of the introduction and then head over to the documentation for the REST API and examine the endpoint for creating agreements.
